PROBLEMA CORRIGIDO
───────────────────────────────────────────────────────────────────────────
Erro: "Table 'finanza_db.usuario' doesn't exist"

Este erro ocorria porque o servidor conectava ao MySQL mas as tabelas não
existiam. Era necessário executar manualmente o script SQL antes de usar.

───────────────────────────────────────────────────────────────────────────

O servidor agora cria automaticamente todas as tabelas necessárias na
primeira execução. Não é mais necessário executar scripts SQL manualmente!

COMO USAR
───────────────────────────────────────────────────────────────────────────

1. Instale e inicie o MySQL Server

2. Crie o banco de dados:
   mysql -u root -p
   CREATE DATABASE finanza_db;
   exit;

3. Configure credenciais (se necessário):
   Edite: ServidorFinanza/src/util/DatabaseUtil.java
   Altere DB_USER e DB_PASSWORD

4. Compile e execute o servidor:
   cd ServidorFinanza
   ant compile
   ant run

5. As tabelas serão criadas automaticamente! 🎉
